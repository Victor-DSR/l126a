A aula 10 foi dedicada à implementação do último exercício acima.

A função lê um número qualquer de caracteres, até ler um *enter* ('\n').
O processamento que será feito com o caractere depende de seu valor: se for um dígito, deve ser usado para compor o número sendo lido; se for *enter*, a execução da função deve ser encerrada e o número lido retornado à função chamadora; se não for nem *enter* nem um dígito, deve ser ignorado.
Podemos esquematizar a função assim:
```
   repete
     lê um caractere
     seleciona o que fazer de acordo com o caractere lido:
     se for um dígito
       usa o dígito no número sendo composto
     se for enter
       retorna o número composto, encerrando a função
     se não for nenhuma das opções acima
       ignora o caractere
```
Pelo esboço acima, dá para ver que:
- a função precisa manter pelo menos dois valores:
  - o do caractere sendo processado,
  - o do número em formação.
- a função tem um laço de repetição, usado para a entrada e o processamento de cada caractere.
- o término do laço é realizado por uma decisão tomada pelos comandos controlados pelo laço (e não no início de cada repetição).
- o processamento de um caractere é uma seleção múltipla (deve ser escolhida uma ação entre várias (no caso, 3) possibilidades).

Os valores vão corresponder a variáveis. No caso do caractere, é um valor difirente a cada execução do laço, um valor que não precisa sobreviver de uma execução do laço para a seguinte — a variável que vai conter esse valor pode ser declarada no interior do laço.
No caso do variável que contém o número que a função está formando, esse valor é passado de uma execução do laço para a seguinte, para calcular o número resultante da adição de um dígito é necessário o número anterior sem esse dígito. Essa variável deve ser declarada fora do laço.

Como a saída do laço é realizada a partir de um valor que é definido na mesma execução do laço, o teste de término não será colocado no cabeçalho do comando do laço (é um laço "infinito" do ponto de vista do cabeçalho do laço).
Com *while* se implementa laço infinito com um teste que nunca falha, como:
   ```c
   while (1) {
       // ...
   }
   ```
Com *for*, pode-se omitir o teste, que será tratado como sempre verdadeiro:
   ```c
   for (;;) {
       // ...
   }
   ```

A seleção múltipla, com a garantia de que exatamente um dos comandos é selecionado, pode ser implementada com comandos *if* aninhados:
   ```c
   if (teste 1) {
       // comandos 1
   } else if (teste 2) {
       // comandos 2
   } else {
       // comandos 3
   }
   ```

Juntando tudo isso, podemos traduzir grande parte do nosso esboço para C:
   ```c
   int lenum() {
     int num = 0;
     for (;;) {
       int c = // lê um caractere
       if (/* c é um dígito */) {
         // usa o dígito no número sendo composto
       } else if (/* c é enter */) {
         // retorna o número composto, encerrando a função
       } else {
         // ignora o caractere
       }
     }
   }
   ```
Para ler um caractere, usamos `getchar`.
Para saber se é enter, usamos um teste simples.
Para retornar o número, usamos `return`.
Para ignorar, usamos um comando vazio, `;`.
   ```c
   int lenum() {
     int num = 0;
     for (;;) {
       int c = getchar();
       if (/* c é um dígito */) {
         // usa o dígito no número sendo composto
       } else if (c == '\n') {
         return num;
       } else {
         ;
       }
     }
   }
   ```
Para saber se um caractere é um dígito, usamos uma função auxiliar:
   ```c
   // retorna 1 se n for o código de um dígito, 0 se não for
   int digito(int n) {
     if (n < '0') return 0;
     if (n > '9') return 0;
     return 1;
   }
   ```
Para usar o dígito cujo código está em *c* para compor o número que está em *num*, extraímos o valor do dígito e acrescentamos ao número (como estava na descrição do exercício):
   ```c
   int lenum() {
     int num = 0;
     for (;;) {
       int c = getchar();
       if (digito(c)) {
         int valor_do_digito = c - '0';
         num = num * 10 + valor_do_digito;
       } else if (c == '\n') {
         return num;
       } else {
         ;
       }
     }
   }
   ```
O comando que não faz nada pode ser retirado (mas continua sendo uma seleção de 3), e colocando uma função *main* para testar:
   ```c
   #include <stdio.h>

   // retorna 1 se n for o código de um dígito, 0 se não for
   int digito(int n) {
     if (n < '0') return 0;
     if (n > '9') return 0;
     return 1;
   }

   // retorna um número inteiro lido do teclado,
   //   ignorando caracteres não dígito
   int lenum() {
     int num = 0;
     for (;;) {
       int c = getchar();
       if (digito(c)) {
         int valor_do_digito = c - '0';
         num = num * 10 + valor_do_digito;
       } else if (c == '\n') {
         return num;
       } else {
         ;
       }
     }
   }

   int maior(int n1, int n2)
   {
     if (n1 > n2) return n1;
     return n2;
   }

   int main()
   {
     int a = 0;
     int b = 0;
     printf("Digite um número ");
     a = lenum();
     printf("Digite outro número ");
     b = lenum();
     printf("Você digitou %d e %d. O maior é %d.\n", a, b, maior(a, b));
   }
   ```
